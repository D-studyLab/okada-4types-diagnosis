# 岡田斗司夫の4タイプ診断

岡田斗司夫氏が提唱する「欲求の4タイプ」理論に基づいた診断ツールです。

## 概要

このアプリは、あなたの行動原理となる本能的な欲求を判定する非公式ファンサイトです。
8つの質問に答えることで、以下の4つのタイプのいずれかに分類されます：

- **司令型**（外交的 × 具体的）- 勝利・序列・合理的
- **注目型**（外交的 × 抽象的）- 情熱・人気・感情的
- **法則型**（内向的 × 具体的）- 自由・分析・仕組み
- **理想型**（内向的 × 抽象的）- 真実・美学・こだわり

## 技術スタック

- React 19
- Vite
- Tailwind CSS
- Lucide React（アイコン）

## 開発

```bash
# 依存関係のインストール
npm install

# 開発サーバーの起動
npm run dev

# ビルド
npm run build

# プレビュー
npm run preview
```

## デプロイ

このプロジェクトはVercelにデプロイすることを想定しています。

### Vercelへのデプロイ手順

1. [Vercel](https://vercel.com)にログイン
2. 「New Project」をクリック
3. GitHubリポジトリをインポート
4. 設定は自動検出されます（`vercel.json`が設定済み）
5. 「Deploy」をクリック
