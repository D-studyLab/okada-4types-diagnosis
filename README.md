# 岡田斗司夫の4タイプ診断

岡田斗司夫氏が提唱する「欲求の4タイプ」理論に基づいた診断ツールです。

## 概要

このアプリは、あなたの行動原理となる本能的な欲求を判定する非公式ファンサイトです。
8つの質問に答えることで、以下の4つのタイプのいずれかに分類されます：

- **司令型**（外交的 × 具体的）- 勝利・序列・合理的
- **注目型**（外交的 × 抽象的）- 情熱・人気・感情的
- **法則型**（内向的 × 具体的）- 自由・分析・仕組み
- **理想型**（内向的 × 抽象的）- 真実・美学・こだわり

## Google AdSense設定

### 1. サイトの所有権確認（完了済み）

`index.html`の`<head>`セクションにAdSenseコードスニペットが追加されています。

### 2. 広告ユニットの作成と配置

1. [Google AdSense](https://www.google.com/adsense/)にログイン
2. 「広告」→「広告ユニット」→「新しい広告ユニットを作成」
3. 広告ユニットを作成すると、**slot ID**が発行されます
4. `src/App.jsx`内の各`AdSenseUnit`コンポーネントの`slot`プロパティにslot IDを設定してください

例：
```jsx
<AdSenseUnit slot="1234567890" format="auto" className="mb-6" />
```

### 3. 広告の配置場所

現在、以下の5箇所に広告ユニットを配置する準備ができています：

- ヘッダー下（イントロ画面）
- 結果表示前
- 結果カード内（シェアボタン下）
- チャートと公式リンクの間
- フッター上

### 4. 審査について

- AdSenseの審査が通るまで、広告は表示されません
- 審査には通常1〜2週間かかります
- 審査が通るまでは、`slot`が空のため広告は表示されません
